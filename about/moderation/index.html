Guide to Moderation

This page provides detailed information on how moderation is done on the Hatch website.

Project Ratings

All projects on Hatch are subject to receiving an age rating. These are provided out manually by the Hatch Team. If a project has not yet been reviewed by a Hatch Team member, it will be be rated N/A.

N/A This project has not been rated yet. Only visitors who are logged in can view this project. Please report inappropriate projects.
Safe This project is suitable for all audiences. This project contains little to no violence and/or language.
Light This project is suitable for most audiences. This project contains little to no violence and/or language, and may receive an NFE rating on Scratch.
Moderate This project is suitable for some audiences. This project may contain some violence, cartoonish blood, light profanity, and scares, and would likely receive an NFE rating on Scratch.
Restricted This project only suitable for teenagers. This project may contain intense violence, blood, profanity, scares, and moderate suggestive content, and would be taken down on Scratch.
